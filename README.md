# 嵌入式小白手册

本项目主要用于个人在学习嵌入式的过程中，整理的相关知识点和踩坑记录

## [项目目录](./toc.md)
+ 记录了当前项目中的所有内容
+ category部分按照8个方面对嵌入式领域的相关概念做了划分，分别是:
    1. 核心硬件组件(Core Hardware Components)
    2. 硬件接口与外设(Hardware Interface and Periphrals)
    3. 通信与调试协议(Communication and Debugging Protocol)
    4. 软件开发工具链(Software Development Toolchain)
    5. 软件架构与层次(Software Architecture and Layers)
    6. 指令集与架构标准(Instruction Set and Architecture Standards)
    7. 调试与测试方法(Debugging an Test Methods)
    8. 应用领域与场景(Application Fields and Scenarios)
