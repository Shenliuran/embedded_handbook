## 功能模块

### DAPlink下载调试
+ 负责芯片: CH549G
+ 使用WCHISPTool工具烧录程序

### 逻辑分析仪
+ 负责芯片：ESP32S3 + RP2040

### 数控电源
+ 负责芯片：ESP32S3 + MP2816

### 简易示波器
+ 负责芯片: ESP32S3 + MCP4017

### 无线下载器 / 串口
+ 负责芯片: ESP32S3

### 串口助手
+ 负责芯片: CH549G (硬件串口) + ESP32S3 (无线串口)
+ UART helper
+ 使用UART helper功能时，应当使用 *ROW 1* 的`RXD`和`TXD`

### 电压表 / 功率计
+ 负责芯片: INA226

### I2C 扫描
+ 负责芯片: ESP32S3

### PWM 输出
+ 负责芯片: ESP32S3

### 数字频率计
+ 负责芯片: ESP32S3
