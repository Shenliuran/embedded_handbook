# 原理图设计规范

## C

### 电容

#### 数码表示法
+ 核心规则：用 2~3 位数字表示容量，单位默认 **pF**；若含字母（如 `104J`），前 2 位为有效数字，第 3 位为 10 的幂次，字母表示误差等级。
+ 误差等级对应：J(±5%)、K(±10%)、M(±20%)
+ 示例：
    - `104J` = 10 × 10⁴ pF = 100000 pF = 0.1 μF，误差 ±5%
    - `223K` = 22 × 10³ pF = 22000 pF = 22 nF，误差 ±10%
    - `330` = 33 pF（无幂次位，直接表示数值）

## R

### 电阻

#### 颜色表示法

![电阻的颜色表示法](../../assets/images/Resistor-Marking.png)

#### 数码表示法
+  **三位数码法**
    + 核心规则：前 2 位为有效数字，第 3 位为 10 的幂次，单位默认 **Ω**；字母表示误差（无字母默认 ±20%）。
    + 误差等级对应：F(±1%)、G(±2%)、J(±5%)、K(±10%)
    + 示例：
        - `104J` = 10 × 10⁴ Ω = 100000 Ω = 100 kΩ，误差 ±5%
        - `221K` = 22 × 10¹ Ω = 220 Ω，误差 ±10%
+  **四位数码法（精密电阻）**
    + 核心规则：前 3 位为有效数字，第 4 位为 10 的幂次，用于表示精度更高的电阻。
    + 示例：
        + `1001F` = 100 × 10¹ Ω = 1000 Ω = 1 kΩ，误差 ±1%

## V

### VCC
+ Voltage Common Collector（集电极公共电源）
+ 最早针对双极型晶体管（BJT） 电路，指给晶体管集电极提供的公共正电源。
+ 实际用法:
    + 传统 TTL 电路、分立元件电路的主正电源
    + 现在也常作为**板级外部供电**、**接口电源**的标识，比如 5V VCC、3.3V VCC
+ 极性：正电源（+）
+ 配对参考：GND

### VDD
+ Voltage Drain Drain（漏极公共电源）
+ 针对MOS 管、CMOS 工艺电路，指给 MOS 管漏极提供的核心正电源
+ 实际用法：
    + 数字芯片（MCU、CPU、FPGA）的内部核心供电
    + 现代嵌入式系统里，VDD 多代表芯片内核 / 数字逻辑的正电源，*常和 VSS 成对出现*
    + 很多场景下**VCC、VDD都指正电源**，可粗略通用，但 VDD 更偏向 “芯片内部供电”
+ 极性：正电源（+）

### VSS
+ Voltage Source Source（源极公共电源）
+ 对应 MOS 管的源极，是 MOS 电路的公共参考地
+ 实际用法：
    + 数字芯片的地（GND），电源负极、零电位参考点
    + 永远和 VDD 配对：**VDD = 数字正电，VSS = 数字地**
+ 极性：地/GND、负极

### VEE
+ 全称：Voltage Emitter Emitter（发射极公共电源）
+ 模拟电路常用的*负电源*，和 VCC/VDD 对应，多用于运放、模拟前端的双电源电路（如 ±5V 中的 -5V）
+ 极性：负电源（-）
