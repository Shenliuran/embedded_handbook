# 主控芯片

## D

### DAP
+ Debug Access Port(调试访问端口)
+ 是 ARM Cortex 系列微控制器（MCU）内部集成的硬件调试接口模块，是芯片层面实现调试功能的核心硬件单元

---

## E

### EEPROM
+ Electrically Erasable Programmable Read-Only Memory(带电可擦可编程只读存储器)

---
## N

### Northbridge
+ 北桥(芯片)
+ 负责与CPU进行高速通信，并连接内存和显卡等高速设备，通常位于CPU插座附近
---
## S

### Southbridge
+ 南桥(芯片)
+ 负责与低速设备(如硬盘、USB等)进行通信，处理I/O管理，通常位于主板的下方

#### RP1
+ Raspberry Pi 1st-party I/O controller
+ 是树莓派基金会自主研发的首款南桥芯片 / IO 控制器，专门为树莓派 5（Raspberry Pi 5） 设计，负责处理所有对外输入输出功能，与 BCM2712 主处理器配合工作
